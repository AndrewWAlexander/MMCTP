Download

We have provided self extracting install files for Windows and Mac OS platforms under method 1. This is the prefered method of installation. Method 2 is for manual installation.

Method 1

Mac OS self-extracting installation wizard v8.9.4 10M
Windows self-extracting installation wizard v8.9.4 20M

Method 2

This method is for Linux, or other platforms that can not run the installation wizard.
  • First download the installation files
    • MMCTP build zip v8.9.7a
    • Configuration folder zip
    • MMCTP manual
  • Follow the platform specific intructions on where to place
    • Mac OS
      1. Move the MMCTP excitable and configurations folder within /Applications/MMCTP/
      2. Install Quesa
    • Windows OS
      1. Move the MMCTP excitable and configurations folder within \Program Files\McGill Medical Physics\MMCTP\
      2. Install Quesa. Place the Quesa.dll file at the location : \Windows\ Quesa.dll
    • Linux OS
      1. Move the MMCTP release folder to the path: /usr/lib/MMCTP/
      2. Move the MMCTP configurations folder to the path: /usr/lib/MMCTP/Configurations
      3. Create the MMCTP settings folder, called “MMCTP-Settings” at the path: /home/yourusername/MMCTP-Settings
