MMCTP Course

The complexity of combining a treatment planning system and established MC codes under one software package is an enormous task and resulted in a complex piece of software. As such, novice users can benefit greatly by attending our 1 day course on MMCTP. The course covers installation, basic MMCTP functionality and aims to provide the user with in-depth information on how all the pieces of MMCTP fix together.

Please contact us if you would like to attend the next course. Course scheduling will depend on user interest.
